Xenoblade Chronicles 3 Checklist

About This Project

Welcome to the Xenoblade Chronicles 3 Checklist, a comprehensive tracker designed to help players stay organized while exploring the vast world of Xenoblade Chronicles 3. This checklist covers everything from collectibles to unique monsters, ensuring that completionists can achieve 100% progress in the game.

Built using Symfony, this project showcases modern web development techniques, offering an intuitive and responsive user interface for fans of the game.

Features

Comprehensive Tracking

Track collectibles, quests, heroes, and more.

Progress Tracking

Monitor progress for every category effortlessly.

Responsive Design

Seamless usage across devices of all sizes.

User Accounts

Personalized tracking through user accounts.

Technologies Used

  • PHP with Symfony Framework
  • HTML5 and CSS3
  • Twig templating engine
  • MySQL for database management
